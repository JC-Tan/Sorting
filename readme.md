# Sorting Algorithm Visualizer

This is a simple webpage that shows how basic sorting algorithms work.

Algorithms:
- Insertion Sort
- Selection Sort
- Merge Sort
- Bubble Sort
- Quick Sort 

You can:
- Change the size of the array
- Change the speed
- Generate a new array to sort

[Demo](https://www.youtube.com/watch?v=v5HnErjZffE)
